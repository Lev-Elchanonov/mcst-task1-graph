Перед сборкой программы надо скачать несколько пакетов: meson, graphviz, gtest

Используется система сборки Meson.
Для сборки проекта надо выполнить команды:
meson setup buildDir
cd buildDir && meson compile
Появится два исполняемых файла: graph и tests
Для запуска основной программы: ./graph
Для запуска тестов: ./tests

Для сборки только основной программы: meson compile graph
Для сборки только тестов: meson compile tests

В программе также присутствует пара дополнительных команд:
LOOK - для отрисовки графа с помощью graphviz и просмотра графа с помощью xdg
CLEAR - для полной очистки графа
